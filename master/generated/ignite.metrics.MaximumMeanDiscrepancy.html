Examples

To use with Engine and process_function, simply attach the metric instance to the engine. The output of the engine‚Äôs process_function needs to be in the format of (x, y). If not, output_tranform can be added to the metric to transform the output into the form expected by the metric.

For more information on how metric works with Engine, visit Attach Engine API.
